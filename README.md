# 🛍️ ALAN WALKER — E-Commerce Web App

A modern, mobile-first e-commerce SPA (Single Page Application) built with pure **HTML, CSS & JavaScript** — no frameworks, no build tools.

## ✨ Features

- 🏠 **Home page** with animated banner slider, deals timer & category grid
- 🗂️ **Categories page** with all product categories
- 🔍 **Live search bar** with glassmorphism design & mic icon
- 🛒 **Shopping cart** with slide-in drawer, quantity controls & order confirmation
- 💳 **Login & Sign Up pages** with glassmorphism card, social login buttons & password toggle
- 📦 **My Orders** page
- ❓ **Help** page
- 👤 **Account** page
- 22 products across 9 categories with real product images
- Fully responsive — works on mobile, tablet & desktop

## 🚀 Getting Started

No install needed. Just open `index.html` in your browser.

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Open in browser
open index.html
```

## 📁 Project Structure

```
├── index.html      # Main HTML — all SPA pages
├── styles.css      # All styling (glassmorphism, animations, layout)
├── app.js          # All JavaScript (routing, cart, products, auth)
└── README.md
```
